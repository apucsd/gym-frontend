## Installation and Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/Nasir1290/gym-client.git
    ```
3. Install dependencies:
    ```bash
    yarn add
    ```

5. Start the application:
    ```bash
    yarn dev
    ```

## Tech Stack

- **Frontend:** React, Next.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **ORM:** Mongoose
- **Validation:** Zod
